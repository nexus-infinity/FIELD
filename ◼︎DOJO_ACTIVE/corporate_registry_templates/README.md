# Corporate Registry Templates

This directory contains the CSV templates and tracking for the BERJAK | NEXUS | INFINITY FRE data ingestion.

## Directory Structure
- `/templates/` - CSV template files with headers
- `INGESTION_STATUS.md` - Status tracking and progress metrics
- `name_mappings.json` - Fuzzy name mapping configurations
