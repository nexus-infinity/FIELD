## Key Visual Elements

### 1. Tonal Ledger (Harmonic Spectrum)
- **Green bars**: Harmonic frequencies (clean transactions)
- **Yellow bars**: Minor dissonance (requires review)
- **Red bars**: Major dissonance (fraud detected)
- **Audio controls**: Play transaction "music" to hear fraud

### 2. Geometric Overlay
- **Complete polygons**: Balanced transaction flows
- **Broken shapes**: Fraudulent or incomplete patterns
- **Real-time drawing**: Watch geometry form/break as transactions process

### 3. Chakra Resonance Map
- **Vertical spectrum**: System integrity visualization
- **Color-coded bars**: Energy flow through different system layers
- **Fraud impact**: Shows which system levels are compromised

### 4. Shadowcasting Lens (★ Framework)
- **Future State (C)**: Desired clean configuration
- **Present Flows (B)**: Current transaction patterns
- **Anchor Truths (A)**: Vault-verified baselines
- **Branch analysis**: Dead-end paths highlight fraud

### 5. Sonic Alert System
- **Real-time audio**: Transaction flows as musical harmony
- **Discord detection**: Fraudulent patterns create jarring sounds
- **Isolation controls**: Focus on specific fraud signatures

## Dashboard Behavior

**Normal Operation**: 
- Smooth geometric completion

**Fraud Detection**:
