## Troubleshooting

### Common Issues

1. **Database path errors**
   - Ensure /Volumes/Akron is mounted
   - Check write permissions on /Volumes/Akron/bear_data/

2. **Module import errors**
   - Activate virtual environment: `source .venv/bin/activate`
   - Install numpy: `pip install numpy`

3. **Extraction script failures**
   - Scripts embed literal "$DATA_DIR" path - may need adjustment
   - Ensure scripts/ directory exists and has execute permissions

4. **Network health below threshold**
   - Run `mirror.automatic_rebalancing()` to identify issues
   - Check energy drains and boundary adjustments
